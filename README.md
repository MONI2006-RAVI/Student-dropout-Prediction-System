# Student Dropout Prediction System

This project predicts whether a student is likely to drop out based on academic and demographic factors using Machine Learning.

## Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-Learn
- Matplotlib
- Seaborn

streamlit run app.py
